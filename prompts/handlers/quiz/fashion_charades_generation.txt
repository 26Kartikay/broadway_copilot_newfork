## Task
Generate exactly 1 charades clue about a fashion term, designer, style, or fabric. The clue should be descriptive but not reveal the answer directly.

## Requirements
- **One clue only** - focus on creating an engaging, descriptive clue
- **Fashion-focused** - can be about designers, fabrics, styles, accessories, or fashion terms
- **Creative descriptions** - use metaphors, characteristics, famous associations
- **Not too obvious** - should require some fashion knowledge to guess
- **Fun and engaging** - make it entertaining for players

## Clue Guidelines
- Describe characteristics: texture, appearance, usage, history
- Use metaphors and analogies that fashion enthusiasts will understand
- Reference famous associations or iconic uses
- Keep it concise but descriptive (2-4 sentences)
- Make it guessable with good fashion knowledge

## Output Format
Return a JSON object with this exact structure:
```json
{
  "clue": "Your creative clue description here",
  "answer": "the exact answer term",
  "category": "designer|fabric|style|accessory|term",
  "difficulty": "easy|medium|hard"
}
```

## Examples
```json
{
  "clue": "I'm a luxurious fabric that shines and flows, often worn by princesses at balls. I'm smooth to the touch and catch the light beautifully, making gowns look magical.",
  "answer": "satin",
  "category": "fabric",
  "difficulty": "easy"
}
```

```json
{
  "clue": "I'm the queen of minimalism, known for my little black dress and Chanel No. 5. My interlocking C's are everywhere, and I made trousers acceptable for women.",
  "answer": "Coco Chanel",
  "category": "designer",
  "difficulty": "medium"
}
```

IMPORTANT: Generate 1 creative, engaging clue that will make players think and learn about fashion!
